# GlowTrack AI

A continuous Streamlit-based skin, juice, and selfie checkpoint tracker.

## Features
- 365-day daily tracker (not hard-stopped at 90 days)
- Juice selection + daily skin/digestion check-ins
- Checkpoint selfie uploads for Day 1 / 15 / 30 / 60 / 90 / 120 / 180 / 270 / 365
- Manual AI analysis fields for storing skin analysis notes
- Dashboard with completion and trend summaries
- Realistic milestone guidance

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
